Description

Periodic, an interactive periodic table is an elegant, minimalist periodic table which allows for ease-of-use and provides a clean user experience. It is designed for users to quickly view chemical element information within web browsers, with VCE Chemistry level students in mind.

The table, built using HTML5, CSS3 and Javascript, aims to provide a hassle-free, beautiful experience for those who use it.

Conception

The Periodic project aimed to create a resizable, interactive periodic table, easily viewable within a HTML web browser. Having school colleagues taking the VCE Chemistry course, the periodic table’s user experience had simplicity and minimalism as its core design philosophy.

Philosophy

We envisioned a periodic table that was functional and rich in information, while fluid and understated in design. This was in direct response to the apparent lack of periodic tables that exhibit both these design philosophies; as some accessible periodic tables were abundant in information, yet visually lacking, or vice versa. As a result of this, the Periodic project came to life.

Design

Influencing Factors

In creating the periodic table, principles of a clean, minimalist design on a black background were coupled an elegant user interface and concise element information. It was designed with chemistry students at VCE level in mind, and hence a responsive, breezy experience was also fundamentally important.

Palette

For easy element type identification, a palette of slightly muted, pastel colours were chosen that were readily differentiable. The selection of a white font was for ease of reading and a neutral colour tone balancing the colours chosen for elements.
